Supervised Learning

Para tener una idea básica de cómo funciona el Supervised Learning (Aprendizaje Supervisado) empezaremos viendo un ejemplo sencillo del mismo y haremos referencia a la notación por la cual esta se maneja.  Supongamos que tenemos un dataset donde se tienen los precios de terrenos de casas con sus respectivos precios.

Los cuales podemos graficar:

Dado este dataset, ¿Cómo podemos aprender a predecir los precios de otras casas, en función al tamaño del terreno? A modo de establecer una notación, usaremos $ x^i $ para hacer referencia a las variables de entrada (en este ejemplo el tamaño del terreno de la casa) tambien llamados como input features (características de entrada), y $ y^i$ para denotar la salida o tambien llamado como target variable, el cual es el que tratamos de predecir (en este ejemplo el precio). El par $(x^i,y^i)$ es llamado training example, y el dataset que usaremos para aprender - una lista de m training examples ${{(x^i,y^i;i=1, ..., m)}}$ -es llamado training set. Nótese que el superscript "(i)" en la notación es un index en el training set, y no así un exponente. Tambien usaremos $X$ para denotar el espacio de los valores de entrada, y $Y$ para el espacio de los valores de salida. En este ejemplo $ X = Y = \mathbb{R} $.

Para describir un problema de Supervised Learning (Aprendizaje Supervisado) de una manera más formal, nuestro objetivo es, dado un training set, para aprender una función $h: X \rightarrow y$ tal que $h(x)$ es un buen predictor para un correspondiente valor de $y$.
